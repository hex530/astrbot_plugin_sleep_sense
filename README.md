# AstrBot Sleep Sense

让 Bot 拥有真实睡眠感知：睡觉、慵懒、熬夜、补觉、吵醒、睡眠周期、做梦。

由 **夕小柠** 指导，**陆渊** 开发。

## 🌟 功能特性

- **真实睡眠状态机**：支持 `清醒`、`慵懒`、`睡觉`、`午休`、`熬夜` 等多种状态。
- **拟人化唤醒机制**：
  - 基于睡眠周期（浅睡/深睡）动态调整唤醒难度（浅睡更易吵醒，深睡需更多消息）。
  - 支持管理员设置紧急警戒词，立刻唤醒。
  - 唤醒后会有“懵逼”状态，回复语气随清醒程度变化。
- **梦境系统**：
  - 自动收集睡前对话素材作为梦境素材。
  - 利用 LLM 生成碎片化、非逻辑性的真实梦境。
  - 支持多种梦境类型：日常碎片、奇异梦境、美梦、噩梦等。
  - 起床时随机浮现梦境记忆，清晰度随缘。
- **自动化作息**：
  - 支持按周设置不同的作息时间。
  - 自动检测管理员空闲状态进入睡眠。
- **临时闹钟**：支持通过自然语言设置临时闹钟，到点自动唤醒并提醒。
- **情绪联动**：支持熬夜疲劳感积累，连续熬夜会表现出不同的疲倦状态。

## 🛠️ 指令列表

| 指令 | 说明 |
| :--- | :--- |
| `/睡眠 状态` | 查看当前睡眠状态、周期阶段、上次睡眠时长及熬夜统计。 |
| `/睡眠 [清醒\|慵懒\|睡觉\|熬夜]` | 手动切换 Bot 的睡眠状态。 |
| `/梦境 列表` | 查看最近生成的 5 条梦境记录。 |
| `/梦境 今晚` | 查看今晚生成的梦境详情。 |
| `/梦境 清除` | 清除所有梦境记录。 |

## ⚙️ 配置项说明

插件支持丰富的 WebUI 配置项，包括：
- **基础设置**：管理员 QQ、各状态下的系统提示词注入。
- **作息计划**：按周设置入睡与起床时间、睡眠波动范围。
- **唤醒逻辑**：各场景下的唤醒消息阈值、睡眠周期乘数、紧急警戒词。
- **梦境设置**：梦境生成概率、记忆留存概率、各类梦境权重。
- **特殊机制**：午休/补觉逻辑、噩梦触发概率、熬夜决策机制。

## 🚀 安装

1. 在 AstrBot 插件管理器中搜索 `astrbot_plugin_sleep_sense` 并安装。
2. 配置管理员 QQ。
3. 重启 Bot 即可开启睡眠感知。

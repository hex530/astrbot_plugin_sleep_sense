# 🌙 AstrBot 睡眠感知插件 (SleepSense)

让你的机器人拥有真实的作息感。它会困、会偷懒、会熬夜、会被吵醒，还会做梦。

## 🚀 安装方法
在 AstrBot 插件管理中，使用仓库 URL 安装：
`https://github.com/hex530/astrbot_plugin_sleep_sense`

## ⚙️ 配置说明
本插件已适配 AstrBot 官方 WebUI 配置。
安装并重载插件后，在插件列表点击 **“配置”** 按钮，即可在网页端直接修改：
- **主人 QQ**：设置谁是它最在意的人（熙熙）。
- **作息时间**：设置睡觉和起床的时间。
- **模式开关**：开启或关闭慵懒模式等。
